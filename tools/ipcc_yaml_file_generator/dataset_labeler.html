Data ingested into the CEDA archive for the IPCC report must be acompanied by metadata to allow searching. Fill in the form below to create a .yaml metadata file to add to your dataset.

Figure number

The number of the figure in the IPCC report

Chapter number

The number of the chapter in the IPCC report

Caption Title

Use the figure title from the assessment report draft provided by the TSU

Caption Text

Use the caption text from the assessment report provided by the TSU

Authors

A list of authors of the final data

Firstname
Surname

Keywords

A list of keywords to be included. Please capitalise keywords and seperate them with commas

The following keywords are automatically added and do not need to be included:

'IPCC-DDC, IPCC, AR7, WG1, WGI, Seventh Assessment Report, Working Group I, Physical Science Basis, Intergovernmental Panel on Climate Change,'

How is the temporal range of the data represented?

You can choose between two options: Common Era (CE) or Before Present (BP)
  • Common Era follows the traditional structure (YYYY-MM-DD) (00:00)
  • Before Present allows for free text entry to describe the temporal range
