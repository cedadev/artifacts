FTP access to ASST products

Raw data for the ASST images are available from our anonymous FTP site. Please note that it is very important that you set your ftp client to use BINARY transfer mode before beginning download.

  1. FTP into ssta.ag.rl.ac.uk (130.246.191.149).
  2. Log on as ANONYMOUS, using your SITE NAME as password
  3. Set to the appropriate directory as follows cd [.asst.{instrument}.{the year}.{the month}] then ftp the files in the usual way

Please note that Internet Explorer has a bug which prevents access to files on a VMS FTP server.

Files on the ftp server are compressed using GZIP. These files can be uncompressed on Unix by Gunzip, and on the PC using Winzip
