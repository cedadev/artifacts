    This site is base on Leonardo, a python web package written by James Tauber.

    The python code has been hacked to include a range of forms for entering images, software, data and documentation into the site with accompanying meta-data.

    Metadata are stored both as python databases, for internal use, and as XML files for portability.

    Pages are stored in a directory tree. A simple navigation tool lists all the pages in the same directory as the current one, and the directories which contain introductory pages.

    To create an introductory page in the directory "mydir", click on New Page and enter "mydir" as the local URL.

    The default wiki engine is from Bryan Lawrence.
