HadCM3 Control example dataset contents

The HadCM3 control dataset has been retrieved from the Hadley Centre as part of COAPEC. A large number of have been retrieved for a 100 year period (2079-2179), while a more restricted set have been downloaded for a 1000 year period (1849-2849).
