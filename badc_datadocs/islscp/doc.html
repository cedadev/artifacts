ISLSCP Documentation

Indexed documentation

Each volume of the CD-rom set contains at least three directories (DOCUMENT, DATA and SOFTWARE) and a AAREADME.CDS file.

  • Volume 1 contains documentation relative to Vegetation, Hydrology and Soils, Snow, Ice and Oceans, Radiation and Clouds, and Near-Surface Meteorology.

  • Volumes 2, 3, 4 and 5 commonly contain the same Near-Surface Meteorology documentation also available on Volume 1.

For a summary description of the data sets in this collection, see the overview article (OVERVIEW.DOC) on Volume 1.

In addition to the data set documentation, Volume 1 contains the peer review of these data (PEER_REV.DOC), fully formatted documentation files (file name .PDF - see the ADOBE directory), and software to decompress the compressed data files, to read the PDF files, and to run Macintosh CD drives.
