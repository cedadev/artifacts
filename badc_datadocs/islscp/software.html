ISLSCP Software


The decompression software provided on these CDs uses the UNIX compression algorithm. It is provided for a wide variety of platforms including Apple Macintosh, DEC VAX running VMS, and IBM PC compatibles running MS-DOS. An IBM 386 and compatibles, or a Mac II and later machine is recommended for improved performance when using these two platforms. No special hardware is required to use this decompression software.

This software has been compiled from several public-access, FTP anonymous sites on the Internet. Updates to this software are available from time to time from those sites. A full list of these sites is given in the AAREADME.CMP file in the SOFTWARE directory on CD Volume 1. For convenience, some of those sites are listed here:

  • MS-DOS -- ftp.cso.uiuc.edu [128.174.5.61]
  •          Mac -- sumex-aim.stanford.edu [36.44.0.6]
  •         VMS -- ftp.sspc.edu [192.107.46.27]
  • VM/CMS -- vmd.cso.uiuc.edu [128.174.5.98]

The SOFTWARE directory on each CD volume contains a decomprs directory and a mac_drvr directory.
