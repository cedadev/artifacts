Project Spaces at the BADC

Project spaces are restricted access disk spaces created for individual projects that require an off-site, shareable version of their operational data. Project spaces employ the BADC's registration system to allow only authorised users to access data. One individual associated with the project is given the responsibility of granting access to other scientists. The aim of providing project spaces is not as a final archive but for an external data store for use during a project, of the order of 100s of GB. The BADC has no interaction with the data other than providing the disk space and FTP access to all users that have been given permission to use the space.

To gain access to a project space, you should first register as a BADC user (if you are already a registered BADC user, skip this step) then you should apply for access to the project space as per instructions in table below.

If you have been given access to a project space then use ftp to connect to proj-ftp.ceda.ac.uk, entering your BADC username and password when requested. Your project space will be located below directory project_spaces.

Access to view and download files may be obtained via your web browser using the URL ftp://username@proj-ftp.ceda.ac.uk (you will be prompted for your password) however to upload files you will need an FTP client. Please note there are known issues with viewing files via Internet Explorer so we advise that another web browser (e.g. Firefox) is used.

For help with FTP please see the BADC FTP guide.

PLEASE NOTE: Project spaces are organised and arranged by the project/campaign PI and participants. Any queries about the content of project spaces should be addressed to the associated PIs.
