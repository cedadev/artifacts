GISST File Format

Data are stored in ASCII.

All values are integers. Temperature values are in Degree Celsius.

  • Monthly mean SST are stored as counts * 100 (in Degree Celsius)
  • counts of stations used are stored as counts * 100
  • IF GISST ice covered, gridboxes are flagged as -1000
  • missing data are set to -32768.


The month and year are stored at the start of each month.

Data Array (360x180).
Item (1,1) stores the value for the 1-deg-area centred at 179.5W and 89.5N.
Item (360,180) stores the value for the 1-deg-area centred at 179.5E and 89.5S.

         | MON | YR  |

         |                                              |
         |(1,90)                                        |
     Equ |                                              |
         |(1,91)                                        |

         |                                              |
         |                                              |
         |                                              |
     90S |(1,180)______________________________(360,180)|
          180W                 0                    180E



GICE File Format

Data are stored in ASCII.

All values are integers.

  • Ice concentrations are stored as tenths of gridbox covered with ice (0-9)
  • Gridboxes completely ice-covered are stored as '*'
  • missing data are set to '-'


The month and year are stored at the start of each month.

Data Array (360x180).
Item (1,1) stores the value for the 1-deg-area centred at 179.5W and 89.5N.
Item (360,180) stores the value for the 1-deg-area centred at 179.5E and 89.5S.

     90S |(1,180)______________________________(360,180)|
          180W                 0                    180E



MOHMATN4 File Format

Data are stored in ASCII.

All values are integers.

  • Max and Min anomalies are stored as temperature anomalies * 100
  • Counts of stations used are stored as counts * 100
  • Missing data are set to -32768.


The month and year are stored at the start of each month.

Data Array (72x36).
Item (1,1) stores the value for the 5-deg-area centred at 177.5W and 87.5N.
Item (72,36) stores the value for the 5-deg-area centred at 177.5E and 87.5S.


          -----------
         |     |     |
         | MON | YR  |

         |                                              |
         |                                              |
         |                                              |
         |(1,18)                                        |

         |(1,19)                                        |
         |                                              |
         |                                              |
         |                                              |
         |                                              |
     90S |(1,36)_________________________________(72,36)|
          180W                 0                    180E
