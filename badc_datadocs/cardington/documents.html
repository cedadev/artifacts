Please click on this link to view the Cardington documentation online, alternatively you may download the documentation in a zip file from the link below.

Cardington Documentation (cardington_documentation.zip)

This zip file contains a basic description of the dataset, the lat / long co-ordinates for the site, a list of the files available, images and detailed explanations of the instruments, a schematic of the data logging process and information about the data processing undertaken at the site.
