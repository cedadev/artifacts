Description of L0 data types

Directory called Filenames starting Description
aurattn AURATTN.CYYYYDDD* AURA attitude files in plain binary format
auratth AURATTH.CYYYYDDD* AURA attitude files in hdf format.
aurephmn AUREPHMN.CYYYYDDD* Ephemeris file in plain binary format
aurephmh AUREPHMH.CYYYYDDD* Ephemeris file in hdf format
hir0eng P2041616{Ax14}YYDDD* HIRDLS L0 engineering data in plain binary format data packets
hir0sci P2041632{Ax14}YYDDD* HIRDLS L0 science data in plain binary format data packets
gbad1s P2041667{Ax14}YYDDD* APID 0967 = Ground-Based Attitude Determination (GBAD) Data for EOS Aura in 1 second intervals including attitude quaternion, attitude rates, spacecraft position/velocity, gyro data and magnetometer data.
gbad4s P2041658{Ax14}YYDDD* APID 0958 = Ground-Based Attitude Determination (GBAD) Data for EOS Aura in 4 second intervals including earth sensor data.
gbad8s P2041659{Ax14}YYDDD* APID 0959 = Ground-Based Attitude Determination Data for EOS Aura in 8 second intervals including solar ephemeris, coarse sun sensor data, gyro drift bias, attitude quaternion update, magnetic torquer drive on time and star tracker data.
auxiliary_files utcpole.dat.YYYYDDD* ECS internal format for daily values of UT1 - UTC and the x and y components of polar motion.
auxiliary_files leapsec.dat.YYYYDDD* ECS internal format for Toolkit for time difference TAI - UT; From 1961 to 1971, approximate equivalent to TAI-UT
204 is the AURA spacecraft ID
the next 4 digit number is the Level0 APID (application id),
following the 14 As is the time in YYDDDHHMMSS.XXX format
files ending .met contain metadata pertaining to the file of the same name (minus the .met)
