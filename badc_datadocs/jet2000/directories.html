JET2000 data directory structure

This is a draft document, and will change with time.

Description

This document outlines the unix directory structure used at the BADC for the JET2000 data archive. The structure reflects conventions adopted by the BADC for other data archives, which are based on ease of navigation and access.

All aircraft data collected as part of the JET2000 campaign can be found under separate flightnumber directories /badc/jet2000/data/[flightnumber].

Surface-based and satellite data are kept in separate source directories e.g. radiosonde, synoptic_station, meteosat and avhrr

The subdirectory structure of /badc/export/doc and /badc/export/software is not yet defined. These will become defined as the need demands.
