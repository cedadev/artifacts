Met Office Northern Hemsisphere MSLP data file format

The MSLP data are in simple ASCII format.

All data are 6-figure integers.

Field headers are "Day, Month, Year" for daily data (from 1881)

Field headers are "Month, Year" for monthly data (from 1873)

Pressures are stored as hectapascals (hPa) * 100

The first row is for 90 deg N and the last for 15 deg N.

         | DAY | MON | YR  |

         |                                              |
     15N |(1,16)_________________________________(36,16)|
