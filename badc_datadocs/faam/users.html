FAAM Data Access Rules

The table below lists the FAAM data sets and, for each of them, defines the communities entitled to access the archived data. Authorised users can apply by clicking on the relevant APPLY link in the table or by following the link to the list of FAAM Campaigns. Application for access to public processed core data must be done only once (following application, access is automatically granted to public processed core data from all FAAM campaigns). Likewise, only one application (by staff involved in test flights) is needed to access both test flight and raw core data.
To apply for access to any of those datasets that require application, you should first register as a BADC user (anyone is allowed to register). If you are already a registered BADC user, skip this step. If you have forgotten your BADC user ID and/or password, please contact the BADC helpdesk at badc@rl.ac.uk.
