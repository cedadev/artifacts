This list includes the providers of the public data sets.

Before making use of any ACSOE public data set, the user is asked to contact its provider.
The name of the person to be contacted appears on the second line of each data file.

An electronic contact is also given in the data file header. If this electronic address is outdated, please refer to the alphabetical list below to access the person's last e-mail, telephone number and address. In case a data provider cannot be reached, please contact Dr. Claire E. Reeves.
