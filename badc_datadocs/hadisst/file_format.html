HadISST SST File Format

Data are stored in ASCII.

All values are integers. Temperature values are in Degree Celsius.

  • Monthly mean SST are stored as counts * 100 (in Degree Celsius)
  • IF SST ice covered, gridboxes are flagged as -1000
  • missing data are set to -32768.


The month and year are stored at the start of each month.

Data Array (360x180).
Item (1,1) stores the value for the 1-deg-area centred at 179.5W and 89.5N.
Item (360,180) stores the value for the 1-deg-area centred at 179.5E and 89.5S.

HadISST ICE File Format

Data are stored in ASCII.

All values are integers.

  • Ice concentrations are stored as as percentage of gridbox covered with ice.
  • missing data (here land gridboxes) are set to '-32768'


The month and year are stored at the start of each month. These data are monthly averages.

Data Array (360x180)
Item (1,1) stores the value for the 1-deg-area centred at 179.5W and 89.5N
Item (360,180) stores the value for the 1-deg-area centred at 179.5E and 89.5S
