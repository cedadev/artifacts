HadGEM1 Output Variables

The following linked pages list the output variables present in selected output files from the control run of the Met Office HadGEM 1 model. These are described using CDML (Climate Data Markup Language), which is an xml dialect designed for use with climate datasets. The names of the variables themselves are described in the lines "float <variable name>(time axis,z axis,latitude axis,longitude axis)".
