GOSTA 7 or Atlas 7 definition.


ATLAS7 describes the collection of data set compiled in January 1993. They are essentially the same data as those used in the GOSTA publication which was circulated in 1990: Bottomley, M. et al. 1990. (See GOSTA Help file).


The surface flux files are taken from work by J. Hsiung, 1986. (See Ref. 7).
The reference period for the climatologies and anomalies in ATLAS7 is 1951 to 1980, except for heat flux data which are for 1949 to 1979.


The datasets used to compile ATLAS7 are:

MOHSST4 (Meteorological Office Historical Sea Surface Temperature) describes Sea Surface Temperature (SST) data which are quality controlled with extreme values removed. These are the data from which the climatology for the normals period was computed.

MOHSST5 describes SST data which have been supplemented with values from the Comprehensive Ocean-Atmosphere Data Set (COADS) and additionally smoothed by spatial interpolation techniques: Parker, D.E. et al, 1994. (See Ref.5).

MOHMAT2N describes Night Marine Air Temperature (NMAT) data which are quality controlled with extreme values removed. The Night period is defined as sunset to sunrise. COADS data are not included in this data set.

Please refer to Atlas-7 Data set names and Content for details of the data files available on the CD-ROM (HDF version).
