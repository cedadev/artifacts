Met Office Special Request data file Headers

  • MD.DWX.DRNL file format for daily weather measurements
  • MD.HWX.HRLY file format for hourly weather measurements including sunshine hours
  • MD.RAIN.DRNL file format for daily rain measurements
  • MD.RAIN.HRLY file format for hourly rain measurements
  • MD.WIND.HRLY file format for hourly mean wind measurements
  • MD.TEMP.DRNL file format for daily temperature extreme measurements
  • MD.SOILT.DRNL file format for daily soil temperature measurements
  • MD.RADT.HRLY file format for hourly and daily radiation measurements


Important notes:
1- The Met Office "now" use Standard International (S.I) Units.
2 - Values of -999 indicate missing data
