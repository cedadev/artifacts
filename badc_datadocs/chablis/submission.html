CHABLIS - Instructions to Data Providers

  • Preliminaries

    To upload your data files on the BADC computer, you will need to have registered as a BADC user and got a password — if you are not a BADC registered user, please register here; if you are a BADC registered user but have forgotten your user ID and/or password, please ask help from the BADC support desk at badc@rl.ac.uk;


  • File structure

    The BADC is encouraging all CHABLIS projects and campaigns to deliver data in  NASA Ames (ASCII) format. 

    Advice on standardising filenames is provided on the file-naming convention page along with a list of common names used for instruments, locations etc. in filenames.


  • Uploading data

    Data providers can upload their data files directly from their home computer into the BADC incoming directory, via the BADC Web based file uploader. This device can also be used to submit software and text files containing metadata or any relevant information on the project deliverables. Please have your BADC user ID and password handy. You will be asked to select "CHABLIS" then the appropriate subdirectory based on the instrument/model name (please refer to the common names list for details).
