CLIMAT Upper Air Standard Isobaric Surface Check Values - Columns in the CUNL table

Description: CLIMAT upper air standard isobaric surface check values for a month and station at a specified upper air pressure level. CLIMAT normals are not subject to quality control, so there are no QC or version number attributes.

For the convenience of users, the CUNL file column headers are available for download in Excel or as a comma separated text file
