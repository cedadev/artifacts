CLIMAT Upper Air Values from standard pressure levels - Columns in the CURL table

Description: This entity contains monthly values at specified pressure levels from CLIMAT upper air reports.

For the convenience of users, the CURL file column headers are available for download in Excel or as a comma separated text file
