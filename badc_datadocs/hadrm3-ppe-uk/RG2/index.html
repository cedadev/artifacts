Further information and additional files

The following additional information is available for this grid:

  • A simple text file of the y-values.
  • A simple text file of the x-values.
  • A CSV file containing the grid box IDs at each point.
  • A CSV file of the real latitude values at each point.
  • A CSV file of the real longitude values at each point.
  • The rotated pole is defined as:
    • grid_mapping_name = rotated_latitude_longitude
    • grid_north_pole_longitude = 198.0
    • grid_north_pole_latitude = 39.25
  • Image files of the grid in various formats (png, jpg)
  • Zipped shapefile of the geometries of the grid including the grid box ids.
