NEU Data File Names

This convention is currently provisional.

NEU data file names are of the form


instrument_location_YYYYMMDD[hh[mm[ss]]]_[extra-]vnn.ext

where
  • fields inside square brackets are optional;
  • instrument, location, YYYYMMDD, hh, mm, ss, extra and ext are defined in the BADC File Name Convention;
  • vnn is the data version number (from v01 up to v99).

When location is a NEU site, it is a two part field of the form sitename-ly where ly is the site level number (l1, l2 or l3), e.g. easterbush-l3.

Examples of valid NEU data file names:

ceh-met_easterbush-l3_20060801_v01.na
ceh-met_easterbush-l3_20060801_30min-v01.na
ceh-met_easterbush-l3_20060801_c1-v01.na
ceh-met_easterbush-l3_20060801_c1-30min-v01.na
