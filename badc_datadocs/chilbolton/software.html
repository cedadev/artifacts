Software for use with Chilbolton data

A description of software available to access Chilbolton data.

NetCDF data into may be read into some popular data analysis packages using the following software:

  • load_nc.m - loads variables and global attributes from a generic NetCDF file into the Matlab workspace. To use it you will need to download the free NetCDF Toolbox for Matlab 5 (Matlab 6 is also supported on some platforms). Contact Robin Hogan if you have any problems.
  • netcdf.zip - to query, load and plot NetCDF radar data in IDL (provided by Charles Kilburn).
  • read_chilradar.pro - to load NetCDF radar data in PV-WAVE (provided by Richard Forbes).
