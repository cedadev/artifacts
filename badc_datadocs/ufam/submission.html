UFAM - Instructions to Data Providers

  • Preliminaries

    Data from UFAM instruments is stored at the BADC in two ways:
    • Where UFAM instruments have been used during intensive measurement campaigns the data is stored together with the rest of the data in the archive for that campaign, for example, data measured during NAMBLEX is kept in the NAMBLEX archive, data from the TORCH campaign can be found in the Polluted Troposphere archive.
    • UFAM instruments also collect data outside of these campaigns- either during smaller projects or as background measurements in between assignments. This data is kept in the UFAM archive.

    These archives are, by their nature, inter-linked and the higher level directories containing UFAM data are visible from within the individual archives. Separate permission is required to access each individual archive.

    To upload your data files on to the BADC computer, you will need
    1. to have registered as a BADC user and got a password — if you are not a BADC registered user, please register here; if you are a BADC registered user but have forgotten your user ID and/or password, please ask help from the BADC support desk at badc@rl.ac.uk;
    2. to have been granted access to the appropriate archive
      — for access to the NAMBLEX archive, please apply here.
      — for access to the Polluted Troposphere archive (for TORCH data), please apply here and see the polluted troposphere web page.
      — for access to the UFAM archive, please apply here.

  • File structure

    The BADC is encouraging all Polluted Troposphere projects and campaigns to deliver data in either NetCDF (binary) or NASA Ames (ASCII) format. NetCDF data should conform to the Climate and Forecasts (CF) Metadata Convention. Information on the inclusion of metadata is also provided.

    Advice on standardising filenames is provided on the file-naming convention page along with a list of common names used for instruments, locations etc. in filenames.


  • Uploading data

    Data providers can upload their data files directly from their home computer into the BADC incoming directory, via the BADC Web based file uploader. This device can also be used to submit software and text files containing metadata or any relevant information on the project deliverables. Please have your BADC user ID and password handy. You will be asked to select the appropriate subdirectory (please refer to the list below).

    / incoming directory ... Campaign or Project name
    /incoming/polluted-tropo/torch/ Tropospheric ORganic CHemistry expt (TORCH)
    /incoming/namblex/ The North Atlantic Marine Boundary Layer EXperiment (NAMBLEX)
    /incoming/ufam/intercomparison UFAM intercomparison experiment
