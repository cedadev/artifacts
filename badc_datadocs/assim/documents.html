Documentation for the Met Office Assimilated data

A list of the documentation files for the Met Office Assimilated data.

Dataset Document

The dataset document giving an overview of the dataset, and giving detailed descriptions of all fields in the files.

The dataset quality document

Gives an outline of the estimated errors and biases for each parameter in the assimilated data files.

Data Format document

Gives a detailed description of the data format (PP format) used by the Met Office for the assimilated data files. This is not strictly necessary, since the file structure can remain hidden if the Met Office-supplied access routines are used.

UKMO New Dynamics

Details changes to the assimilation system.
