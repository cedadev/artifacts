Met Office Geopotential Height data file format

The Height data are in simple ASCII format.

All data are 6-figure integers.

Field headers are "Day, Month, Year" for daily data.
Field headers are "Month, Year" for monthly data.

Both height and thickness are in metres *100

The first row is for 90 deg N and the last for 15 deg N. 


          ------------------
         |     |     |     |
         | DAY | MON | YR  | 
         |_____|_____|_____|_____________________________
     90N |(1,1)                                         |
         |                                              |
         |                                              |
         |                                              |
         |                                              |
     15N |(1,16)_________________________________(36,16)|
          180W                 0                    180E
