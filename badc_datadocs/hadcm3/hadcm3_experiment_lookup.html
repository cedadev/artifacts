HadCM3 Experiment-Runid Lookup Table

This page shows the relationship between the expeiments conducted with the HadCM3 model and the actual runid's which are used to identify the actual model output when the model is run. Often, an experiment will be made up of several runid's. Note that some of the QUMP data have not yet been added to the BADC archive.
