This diagram shows the simulated dates of HadCM3 runs. Dates appearing on the same horizontal line are at the same point since the start of the HadCM3 spinup, which is shown in the left-hand column. Runs in the same vertical line succeed one another, or run in parallel using the same dates. Some scenario runs shown as ending in 2099 in fact might have gone on to 2100 or 2101. All dates are 1st December.
