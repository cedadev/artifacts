Global Radiosonde (1997 - 2006)- Simple ASCII Data File Format

The radiosonde data is in a simple ASCII format.

-9999999. indicates a missing value.

The first line of the data file contains the following information:

PARAMETERFIELDS
WMO Number1
Date (YYYYMMDD)2
Hour (HH)3
Latitude4
Longitude5
Station Height (metres)6

Headers are indicated on the second line of the data file as follows:

PP     HT     TT     TD     DD     FF

The fields in each file are as indicated below ...

PARAMETERFIELDSCOLUMNSS.I UNITS
PP = Pressure11- 9Pa
HT = Height above sea level210-18metres
TT = Dry-bulb temperature319-28deg K
TD = Dew-point temperature429-38deg K
DD = Wind direction739-47degree true
FF = Wind speed848-57m/s

The data starts on the third line.
