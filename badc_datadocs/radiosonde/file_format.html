European Radiosonde (1990 - June 1998)- Data file format

The radiosonde data is in a simple ASCII format.

-32768 indicates a missing value.

Headers are indicated on the first line of the data file as follows:

PP     HT     TT     TD     RH     HMR     DD     FF

The fields in each file are as indicated below ... 


PARAMETER	                 FIELDS   COLUMNS         UNITS   
--------------------------------------------------------------------
PP  = Pressure                     1        1- 6      millibars x 10
HT  = Height above ground	   2        7-12      metres
TT  = Dry-bulb temperature	   3       13-18      deg C x 10                 
TD  = Dew-point temperature	   4       19-24      deg C x 10
RH  = Relative humidity	  	   5       25-30      %
HMR = Humidity mixing ratio 	   6       31-36      cg/kg
DD  = Wind direction		   7       37-42      degree true
FF  = Wind speed		   8       43-48      knots
--------------------------------------------------------------------

The data starts on the second line.
