FREE - Instructions to Data Providers

  • Preliminaries

    To upload your data files on the BADC computer, you will need
    1. to have registered as a BADC user and got a password — if you are not a BADC registered user, please register here; if you are a BADC registered user but have forgotten your user ID and/or password, please ask help from the BADC support desk at badc@rl.ac.uk;
    2. to have been granted access to the FREE archive — if you do not yet have access to the FREE archive, please apply here.

  • File structure and naming

    The BADC is encouraging all FREE projects to deliver data in either NetCDF (binary) or NASA Ames (ASCII) format. To ensure the future usability of the data, it is important to include all necessary information about the data with the file. This is termed metadata. The data files submitted should conform to the Climate and Forecasts (CF) Metadata Convention. If you wish to check that files are correctly formatted before uploading them please use the NetCDF format checker or NASA-Ames format checker.

    Advice on standardising filenames is provided on the file-naming convention page along with a list of common names used for instruments, locations etc. in filenames.

  • Uploading data

    Data providers can upload their data files directly from their home computer into the BADC incoming directory, via the BADC Web based file uploader. This device can also be used to submit software and text files containing metadata or any relevant information on the project deliverables. Please have your BADC user ID and password handy.
