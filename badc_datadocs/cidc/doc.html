CIDC Documentation

Indexed documentation

Each volume of the CD-ROM set contains four directories (DATA, READMES, SOFTWARE, WWW), a AAREADME file, and a portable document format (PDF) CD booklet.

  • Volume 1 contains documentation regarding the Biosphere, Measured Variable Atmospheric Constituents, Measured Surface Temperatures and Pressures, Hydrological Data, and Angular Radiation Distribution Model.
  • Volume 2 contains documentation regarding the 4-D Atmospheric Dynamics Products.
  • Volume 3 contains documentation relating to Radiation and Clouds.
  • Volume 4 contains documentation relevant to TOVS Atmospheric Soundings.

For a summary description of the datasets in this collection, see the overview article (OVERVIEW.DOC) on Volume 1.
