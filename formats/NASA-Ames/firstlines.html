NASA Ames Format: Header Top Lines

General format

All NASA Ames formatted files have the same first seven lines (header top):

NLHEAD FFI Line 1
ONAME Line 2
ORG Line 3
SNAME Line 4
MNAME Line 5
IVOL NVOL Line 6
DATE RDATE Line 7

  • NLHEAD, FFI, IVOL, NVOL are integers.
  • ONAME, ORG, SNAME, MNAME are character strings of maximum 132 characters.
  • DATE, RDATE are dates in the format YYYY MM DD (with 1 blank between YYYY & MM and between MM & DD). If the first digit of MM or DD is 0, it can be replaced by a blank but its space may not be deleted.

There may be any number of blank spaces (at least one) between

  • NLHEAD & FFI
  • IVOL & NVOL
  • DATE & RDATE
but each of the corresponding lines (1, 6 & 7) must not extend beyond row 132.

Definition of symbols

NLHEAD Number of lines in file header
FFI File format index
ONAME List of author(s) in the format Lastname, Firstname separated by an arbitrary character (for example, a hyphen or a semi-colon).
ORG Organisation name (university, institute, etc). May include address and phone numbers.
SNAME Source of data, i.e. instrument, platform, model name, etc.
MNAME Name of mission, campaign, programme and/or project.
NVOL Total number of files belonging to the considered dataset (i.e. with same ONAME, ORG, SNAME, MNAME).
IVOL Number of the file in the above dataset (between 1 and NVOL).
DATE Universal Time date at which the data within the file begin.
RDATE Universal Time date at which the data within the file have been reduced or revised.

Example

27 1001
Morgan, Caroline; Smith, John
University of Moreton-in-the-Marsh, 32, Forest Road, YZ55 0AB, UK, +44 01234 567890
X1000 Interferometer onboard FAST Jet
INTERESTING Programme / Project No 2259
1 5
1999 06 18  2000 08 03
