IDL code for reading NASA-Ames format files


This page contains links which allow you to view and download IDL code for reading NASA-Ames format files. IDL Stands for "Interactive Data Language" and is a popular language for scientific analysis and visualisation.

The IDL code provided here was written by T. Atwater at NASA GSFC.

Please note that the BADC is providing this code 'as is' and we cannot guarantee that either the documentation or the comments in the code are correct. A BADC user has however pointed out that there is a discrepancy between the documentation and the IDL code comments regarding the date of mission format. In this case, the IDL code seems to have incorrect comments.
