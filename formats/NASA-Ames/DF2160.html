NASA Ames Data Format: FFI 2160

Symbols used

Xs sth independent variable (s = 1 or 2).
X1 is the bounded independent variable. It is a real number.
X2 is the unbounded independent variable. It is a character string.
To each value of X2 corresponds a different set of values taken by X1.
NX Number of values of X1. It is X2-dependent.
If NX(m) = 0, there is no record of X1 and Vn at the mark X2(m).
NV Number of primary dependent variables
NAUXV Number of auxiliary dependent variables
NAUXC Number of auxiliary variables which are character strings.
If NAUXC = 0, then no auxiliary variable is a character string.
Vn nth primary variable
Aa ath auxiliary variable (depends on X2 only — independent of X1).
A1 = NX (integer).
For 1 < a < NAUXV-NAUXC+1, Aa is real.
For NAUXV-NAUXC < a < NAUXV+1, Aa is a character string.

Condensed notation

[ X2(m) ]
[ NX(m) ( Aa(m) , a=2,NAUXV-NAUXC ) ]
[ Aa(m) ] a=NAUXV-NAUXC+1,NAUXV
[ X1(i,m) ( Vn(i,m) , n=1,NV ) ] i=1,NX(m) Omitted if NX(m) = 0 or if NX(m) = AMISS1
