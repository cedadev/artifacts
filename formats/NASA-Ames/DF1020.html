NASA Ames Data Format: FFI 1020

Symbols used

X Independent variable marks (explicit values of the independent variable). Real numbers.
DX Independent variable interval
NVPM Number of independent variable intervals between 2 successive independent variable marks:
X(m+1) = X(m) + NVPM*DX (See Note below)
NV Number of primary dependent variables
NAUXV Number of auxiliary dependent variables
Vn nth primary variable
Aa ath auxiliary variable. Its values are real and are recorded only for explicit values of the independent variable, i.e. at the independent variable marks)

Note: The actual independent variable values at which primary variables Vn are recorded range through
Index of Vn
X(1) First independent variable mark (explicit) 1
X(1) + DX Implicit 2
... ... ...
X(1) + (NVPM-1)*DX Implicit NVPM
X(2) = X(1) + NVPM*DX Second independent variable mark (explicit) NVPM+1
X(2) + DX = X(1) + (NVPM+1)*DX Implicit NVPM+2
... ... ...
X(2) + (NVPM-1)*DX = X(1) + (2*NVPM-1)*DX Implicit 2*NVPM
... ...
X(m) = X(1) + (m-1)*NVPM*DX Independent variable mark number m (explicit) (m-1)*NVPM+1
X(m) + DX = X(1) + [(m-1)*NVPM+1]*DX Implicit (m-1)*NVPM+2
... ... ...
X(m) + (NVPM-1)*DX = X(1) + (m*NVPM-1)*DX Implicit m*NVPM
... ...

Condensed notation

[ X(m) ( Aa(m), a=1,NAUXV ) ]
[ Vn(i), i=1+(m-1)*NVPM,m*NVPM ] n=1,NV

Expanded notation

X(1)A1(1)...ANAUXV(1) 1st Independent variable mark & auxiliary variables values at that mark
V1(1)...V1(NVPM) Values of V1 at X(1), X(1)+DX, ..., X(1)+(NVPM-1)*DX
V2(1)...V2(NVPM) Values of V2 at X(1), X(1)+DX, ..., X(1)+(NVPM-1)*DX
...
VNV(1)...VNV(NVPM) Values of VNV at X(1), X(1)+DX, ..., X(1)+(NVPM-1)*DX
X(2)A1(2)...ANAUXV(2) 2nd Independent variable mark & auxiliary variables values at that mark
V1(NVPM+1)...V1(2*NVPM) Values of V1 at X(2), X(2)+DX, ..., X(2)+(NVPM-1)*DX
V2(NVPM+1)...V2(2*NVPM) Values of V2 at X(2), X(2)+DX, ..., X(2)+(NVPM-1)*DX
...
VNV(NVPM+1)...VNV(2*NVPM) Values of VNV at X(2), X(2)+DX, ..., X(2)+(NVPM-1)*DX
...
