Verify NASA-Ames file format

Run NASA-Ames format checking program

This program can be used to check your NASA-Ames files for correctness. When you submit a file to the BADC it will be verified using this program and will be rejected if it fails.

Help on formatting problems gives help on common formatting problems and how to interpret the output from the format checking program.

The NASA-Ames format specification listed below is also very helpful for understanding the error messages produced by the format checking program.

Submit NASA-Ames files to the BADC

The following links allow you to submit NASA-Ames files to the BADC. This is the preferred method of submitting files to us, rather than using FTP. It will verify your file before submission and will reject any file which fails the verification.

In order to upload files you must have an account name and password with the BADC. This is the same account name and password that you would use to FTP files.
