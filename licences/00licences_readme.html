Introduction

This folder contains data licences prepared by the Centre for Environmental Data Analysis (CEDA). The contents include 4 generic licences designed to aid wider re-used of these licences (as described below) and specific data licences. Not all licences here are in present use.

Generic Licences

4 generic licences have been produced by CEDA to address common licence requirements. These have been based on the UK Open Government Licence (OGL) and UK Non-Commercial Open Government Licences, which are equivalent to CC-by and CC-by-nc. Each of these 4 generic licences has a symlink below which points to the latest version of the licence. All versions of the licence are available under the associated "_versions" sub-folder to aid access to previous versions of the licence or linking to specific versions.

  • CUGL - Closed-Use General Licence: this is a permissive licence that allows for a wide range of reuse of the information resource but differs from open licences by preventing onward sharing of the resource.
  • CUNCGL- Closed-Use Non-Commercial General Licence: this is a permissive licence that allows for a wide range of non-commercial reuse of the information resource but differs from open licences by preventing onward sharing of the resource.
  • RUGL - Restricted-Use General Licence: this licence is to allow a wide range of use of an information resource, but within a limited time period and with additional restrictions requiring permission from the data owner.
  • RUNCGL- Restricted-Use Non-Commercial General Licence: this licence is to allow a wide range of non-commercial use of an information resource, but within a limited time period and with additional restrictions requiring permission from the data owner.

Specific Licences

These licences are used with various resources offered by CEDA either now or in the past.

Additional items

  • depositors_agreement.html - this is the agreement that all data depositors should agree to before depositing data for ingestion to the CEDA Archive
  • missing_licence.pdf - this default 'licence' is used where the licence for a given resource has not yet been determined but an entry is required to support CEDA services.
